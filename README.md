# 🎯 Своя игра — панель ведущего
### (Quiz Show — Jeopardy-style Game Board)

Интерактивное игровое поле для проведения викторины в формате **«Своя игра»**. Инструмент для ведущего: открывайте вопросы, запускайте таймер, показывайте ответы и начисляйте очки игрокам — всё в одном окне браузера.

🔗 **Открыть:** [uipixel.github.io/svoya.igra](https://uipixel.github.io/svoya.igra/)

---

## О проекте

Это панель управления для ведущего живой викторины. На экране — игровое поле с темами и вопросами разной стоимости. Ведущий выбирает вопрос, зачитывает его участникам, при необходимости запускает таймер и проигрывает аудио или показывает картинку, а затем открывает правильный ответ и начисляет (или снимает) очки игроку.

Идеально подходит для вечеринок, корпоративов, уроков и семейных праздников.

## Возможности

- 🗂️ **Игровое поле** из 5 тем по 5 вопросов в каждой (стоимость от 100 до 500)
- 🎵 **Аудиовопросы** со встроенным плеером (воспроизведение, пауза, стоп, повтор)
- 🖼️ **Вопросы с картинками** с возможностью открыть изображение на весь экран
- ⏱️ **Таймер на ответ** (по умолчанию 60 секунд) — запуск и остановка одной кнопкой
- 👁️ Кнопка **«Показать ответ»** — правильный ответ открывается только когда нужно ведущему
- ✅ **Засчитать / не засчитать ответ** — очки автоматически прибавляются или вычитаются
- 🧮 **Учёт игроков**: добавление участников, редактирование имён, ручное изменение счёта, подсветка активного игрока
- 🔄 **Сброс очков** для новой партии
- 🟩 Использованные вопросы помечаются на поле (правильно/неправильно), чтобы не запутаться

## Темы по умолчанию

История · Наука · Музыка · Картинки · Кино

Вопросы и ответы заданы прямо в коде (массив `gameData` в `index.html`), так что вы легко можете заменить их на свои.

## Как пользоваться

1. Откройте игру в браузере (лучше на большом экране или проекторе).
2. Добавьте игроков и при необходимости переименуйте их.
3. Нажмите на ячейку с вопросом — откроется карточка с заданием.
4. Зачитайте вопрос участникам, при необходимости запустите таймер, проиграйте аудио или покажите картинку.
5. Нажмите **«Показать ответ»**, чтобы проверить.
6. Засчитайте или не засчитайте ответ — очки игрока обновятся автоматически.
7. Когда все вопросы сыграны — побеждает игрок с наибольшим счётом! 🏆

## Как заменить вопросы на свои

Откройте `index.html` и найдите массив `gameData`. Каждая тема выглядит так:

```javascript
{
    topic: "Название темы",
    questions: [
        { price: 100, question: "Текст вопроса?", answer: "Правильный ответ" },
        // ...
    ]
}
```

Для вопроса с картинкой добавьте поле `image: "имя_файла.jpg"`, для аудио — `audio: "имя_файла.mp3"` (файлы положите рядом с `index.html`).

## Технологии

Чистый **HTML, CSS и JavaScript** — без фреймворков и сборки. Весь проект в одном файле `index.html`, иконки через Font Awesome.
