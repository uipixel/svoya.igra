# 🎄 Подготовка к Новому году

Интерактивная браузерная игра-квест: решайте головоломки и ребусы, собирайте цифры секретного кода и получите новогодний приз!

🔗 **Играть онлайн:** [uipixel.github.io/happy.new.year](https://uipixel.github.io/happy.new.year/)

---

## О чём игра

Вы попадаете в свою цифровую комнату, где спрятаны **пять предметов-загадок**. Нажимая на каждый из них, вы получаете головоломку, которую нужно решить. За каждую разгаданную загадку вы получаете цифру — часть итогового кода.

Когда все пять головоломок решены, вы вводите собранный код целиком и забираете подарок. Цель — подготовиться к Новому году, разгадав все загадки. 🎁

## Как играть

1. Откройте игру по ссылке выше.
2. Изучите вступление и переходите в комнату с пятью предметами 🧦 🎁 ❄️ 💡 🎄
3. Нажимайте на предметы и решайте головоломки — где-то нужно выбрать правильный вариант ответа, а в одной из загадок ответ вводится вручную.
4. Запишите цифры, которые открываются за каждую решённую загадку.
5. Соберите все пять цифр, введите итоговый код и получите приз!

## Особенности

- 🧩 Пять разноплановых головоломок: логика, наблюдательность, математика и ребусы
- ✅ Загадки с вариантами ответов и одна с ручным вводом и валидацией
- 🔢 Каждая загадка даёт цифру для итогового секретного кода
- 💾 Прогресс сохраняется — можно закрыть игру и вернуться позже, не проходя заново
- 💡 У каждой загадки есть подсказка, если что-то не получается
- 📱 Адаптивный интерфейс — удобно играть и на телефоне, и на компьютере

## Технологии

Игра написана на чистом **HTML, CSS и JavaScript** — без фреймворков и сборки. Весь проект умещается в одном файле `index.html`, поэтому работает быстро и не требует установки.

## Запуск локально

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/uipixel/happy.new.year.git

# 2. Перейдите в папку
cd happy.new.year

# 3. Откройте index.html в браузере
```

Можно просто дважды кликнуть по `index.html` — игра откроется в браузере.
